Windows 桌面 AI 角色寵物

把喜歡的角色，
放在電腦桌面上一起聊天

DesktopST（桌友）是一款可愛的桌面小程式。 填入 API Key 後，就能和自訂的 AI 角色在桌面上對話—— 像是養了一個會講話、會提醒你事情的小夥伴。

下載程式 新手教學

這是什麼？

用三句話講完。

1

桌面上的小角色

角色站在你的桌面上，不像聊天網站要開瀏覽器、登入帳號。點一下就能聊。

2

用 AI 跟你對話

填入 OpenAI、Gemini、Claude、Grok 等其中一家的 API Key 即可開始聊天，角色會用你設定的個性回應。

3

什麼都能改

角色的圖片、個性、世界觀、桌面顏色，全部都可以自己換。

有什麼功能？

六個重點，照著看就懂。

01 · 開始對話

填入 API Key，馬上開聊

支援 OpenAI / Gemini / Claude / Grok，把 API Key 貼進設定畫面就完成。

不知道什麼是 API Key、要去哪裡申請？看 API Key 申請教學。

02 · 自訂角色

角色圖片、個性、世界觀都能改

幫角色換上自己畫的圖、自己寫的人設，甚至寫一份「世界觀設定」讓角色照著演。也支援匯入 SillyTavern 的 PNG 角色卡。

03 · 自由擺放

縮放、移動、水平翻轉

角色可以拖去桌面任何位置、放大或縮小到你喜歡的大小，也能左右翻轉讓他看向另一邊。多個角色可以同時出現在桌面上。

04 · 提醒功能

AI 用角色的語氣提醒你

設定每天的固定時間、或是 N 分鐘後，讓角色主動跳出來說話。提醒的口氣會跟著角色個性走——不是死板的鬧鐘，而是「他」在叫你。

05 · 便利貼

貼在桌面的便利貼，AI 看得懂

可以在桌面上貼好幾張小便利貼（待辦、心情、靈感都可以）。打開選項後，AI 還會根據便利貼的內容自動找話聊。

06 · 多種色系

介面和便利貼都可以換顏色

內建九種色彩主題，喜歡薄荷綠、奶油黃、薰衣草紫、還是深色模式都可以一鍵切換。便利貼也能各自換色，桌面變成自己的小花園。
